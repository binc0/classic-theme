# Minimal Classic PrestaShop Theme

A minimal, modern PrestaShop theme inspired by movity.it website design principles, built with Tailwind CSS for intuitive navigation and clean aesthetics.

## Features

- **Modern Minimal Design**: Clean, contemporary layout inspired by modern Italian design principles
- **Tailwind CSS**: Utility-first CSS framework for rapid development and consistent styling
- **Intuitive Navigation**: Clean, accessible navigation system with hover effects and transitions
- **Responsive Layout**: Mobile-first design that works perfectly on all devices
- **Modern Product Cards**: Clean product grid with hover effects and modern typography
- **Optimized Performance**: Lightweight, efficient CSS and JavaScript
- **PrestaShop 8.x Compatible**: Built for PrestaShop 8.0.x and above

## Installation

1. Download the `minimal-classic-theme.zip` file
2. In your PrestaShop admin panel, go to Design > Theme & Logo
3. Click "Add new theme" 
4. Upload the `minimal-classic-theme.zip` file
5. Activate the theme

## Technical Details

- **Framework**: Tailwind CSS 3.x
- **Font**: Manrope (Google Fonts)
- **Compatible with**: PrestaShop 8.0.x and above
- **Browser Support**: Modern browsers (Chrome, Firefox, Safari, Edge)

## File Structure

The theme follows PrestaShop's standard theme structure:
- `templates/` - Smarty template files
- `assets/css/` - Compiled CSS files  
- `assets/js/` - JavaScript files
- `config/theme.yml` - Theme configuration

## Customization

The theme uses Tailwind CSS classes throughout the templates, making it easy to customize:
- Colors can be modified in the Tailwind config
- Spacing and typography follow Tailwind conventions
- Components use utility classes for easy modification

## Development

To modify this theme:
1. Install dependencies: `npm install` in the `_dev` folder
2. Make changes to the SCSS/template files
3. Build with: `npm run build`
4. The compiled assets will be generated in the `assets/` folder

## Support

This theme is based on the official PrestaShop Classic theme and follows all PrestaShop development guidelines and best practices.
